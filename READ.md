# Sentiment Analysis on Twitter Data

## Objective
This project predicts whether a tweet has positive or negative sentiment using Machine Learning and NLP.

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- NLTK
- TF-IDF Vectorization

## Machine Learning Algorithm
- Logistic Regression

## Workflow
1. Data Collection
2. Data Cleaning
3. Text Preprocessing
4. Feature Extraction using TF-IDF
5. Model Training
6. Evaluation
